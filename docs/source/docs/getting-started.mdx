# Getting Started

seccheck is a Python package with zero external dependencies. It runs on Python 3.9+ and uses only the standard library for all network checks.

## Install from Source

```bash
git clone https://github.com/your-org/seccheck.git
cd seccheck
pip install -e .
```

## Quick Scan

Scan a single URL:

```bash
PYTHONPATH=src python3 -m seccheck scan https://example.com --format text
```

Example output:

```
URL: https://example.com
Final: https://example.com/
Verdict: SAFE  Safety: 9/10  Risk: 5/100  Agent allowed: yes
Findings: none
```

## Use the Gate

The gate command extracts URLs from any text and returns a JSON decision:

```bash
PYTHONPATH=src python3 -m seccheck gate https://example.com --format json
```

## Use the Middleware

The middleware command reads stdin and returns a JSON allow/block decision:

```bash
printf 'check https://example.com' | ./scripts/seccheck-gate
```

## Exit Codes

| Code | Meaning |
|------|---------|
| `0` | All scanned links are allowed |
| `2` | At least one link is blocked by policy |
| `1` | Scanner usage or runtime failure |
